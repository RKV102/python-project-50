# Вычислитель отличий

## Описание проекта

Проект посвящён разработке CLI-утилиты, которая сравнивает содержимое JSON- и YAML-файлов.
Всего доступно три формата представления результатов сравнения:
1. stylish (выбран по умолчанию)
2. plain
3. json
